# Library Management System (Full Stack)

A full-stack Library Management application built using:

## Tech Stack
- Frontend: React.js + Material UI
- Backend: FastAPI
- Database: MySQL
- Authentication: JWT

## Project Structure
|-----backend
|-----frontend



## Features
- User Registration & Login
- Role-based access (Admin, User)
- Book management
- Category management
- Secure authentication

## How to Run

### Backend
```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload


###  Frontend
cd frontend/library-frontend
npm install
npm start
